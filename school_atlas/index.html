Where do Viennese children continue their studies after attending a Volksshule?

A rough analysis using the data of Statistik Austria

The data in the table below are taken from the report of Statistik Austria for the year 2022/23.


Most of the tables and additional information from Statistik Austria are provided in the Report "Schule, Schulbesuch 2022/23" at this link.


In the tables in that report there are plenty of interesting data about the number of children attending various types of schools, divided by Gemeinden (in Vienna: Bezirks), but I could not find any table listing where children continue their studies (normally for the 5th class) after attending the various elementary schools (Volksschulen) in Vienna. These data are instead only provided in the atlas at this link, which is however cumbersome to use since one has to click on every individual school to get this type of information. The table below is generated using data from that atlas, but displaying them in a simpler form. It is likely that similar analysis have been done already also by other people, I do not claim originality :-) and I would be happy to know about other similar attempts.


The next report with the data of the school year 2023/24 is expected to be published on 29.11.2024; it is however not clear if the new data needed for generating the table below will be automatically updated in the atlas at the same time, or only in the next few weeks. If I will have some time (and if the data are still available in the same format), I will update this website accordingly, but I cannot promise anything at the moment.


Some short explanations about the table below:


(1) In the atlas of Statistik Austria there are some movements of students listed as "≤ 6"; in the actual data those movements are instead all equal to zero, and the atlas is manually coded in order to display "≤ 6" in these cases, even if this sounds completely useless. For this reason, I opted for trusting the actual source of data and not their visual representation in the atlas: all these cases are treated as zero.


(2) In the table below you can see some movements of students from some Volksshulen back again in Volksschulen. This is the case for the Wiederholung der Schulstufe im gleichen Schultyp (not necessarily in the same school).


(3) Some schools with the exact same address appear twice in the data of Statistik Austria, hence also in the table below. This is most likely due to the fact that some Viennese schools switched in the last years from a Halbtagsmodell to a Ganztagsschule, thus appearing as two different entities in the data of Statistik Austria.


(4) The next table displays correctly only on a medium/large screen: either use your mobile phone in horizontal mode, or switch to a computer/tablet.
